# Student Performance Prediction using Machine Learning

## 📌 Project Overview

This project predicts student exam performance using Machine Learning. It demonstrates the complete machine learning workflow, including data preprocessing, feature scaling, model training, evaluation, and comparison of multiple classification algorithms.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Handling Missing Values
- Feature Scaling
  - StandardScaler
  - MinMaxScaler
- Model Training
- Model Evaluation
- Accuracy Comparison
- Precision, Recall & F1-Score
- Confusion Matrix

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📂 Project Files

- Student_Performance_Prediction.ipynb
- Student_Performance_Prediction.csv
- requirements.txt

---

## 📊 Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest

---

## 📈 Results

The project compares multiple machine learning models using different evaluation metrics to identify the best-performing model for predicting student performance.
